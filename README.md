Очень простая программа для авторизации и получения кода Telegram в SMS.

Для Mac - скачать бинарь и запустить с зажатым Ctrl.
Для Windows - cкачать exe-шник и запустить.

Если аккаунт не был зарегистрирован, после ввода кода программа предложит ввести First name и Last name. First name необходимо ввести, Last name можно пропустить.

Вопросы/баги/предложения - whattfkk@yandex.ru.
